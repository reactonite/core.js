Class: ReactCodeMapper

ReactCodeMapper(src_dir, dest_dir, props_map)

new ReactCodeMapper(src_dir, dest_dir, props_map)

Class to convert tags and props from HTML to React Call getReactMap method for converting tags fed for HTML and get corresponding React Mapping. Here's an usage example: reactCodeMapper = ReactCodeMapper(source_dir, destination_dir, props_map) react_map = reactCodeMapper.getReactMap(tag_with_attributes) print(react_map)
Parameters:
