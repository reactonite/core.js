About Reactonite

Reactonite is a wrapper for React which allows one to write vanilla html code and convert it to a react code and hence building a PWA, SPA âœ¨
It facilitates one to transpile HTML code to React website, create PWAs and React applications as quickly as possible. Acting as a wrapper to NPM it helps you install dependencies and has functionality of Hot Reloading.
